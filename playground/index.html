Why HOLEXA?

Everything you need in a modern programming language

⚡

Lightning Fast

Compiled to native code using our C-based compiler pipeline. Zero overhead runtime.

🛡️

Memory Safe

No null pointer dereferences, no buffer overflows. Safe by design like Rust.

📖

Readable Syntax

Clean, Python-inspired syntax. If you know Python, you already know HOLEXA.

🌐

Runs Everywhere

Linux, Mac, Windows, iPad, Android. Same code runs on every platform.

📦

Rich stdlib

15+ built-in libraries: math, http, fs, json, crypto, time, random and more.

🔧

Great Tooling

hlxfmt formatter, hlxlint linter, hlxbuild builder, hlxpm package manager.

🎨

Editor Support

VS Code, Vim, Neovim, Emacs, Sublime, IntelliJ — full syntax highlighting.

🐉

Self-Hosting

HOLEXA compiler is written in HOLEXA itself. True language bootstrapping.

Code Examples

See what HOLEXA looks like

// Hello World in HOLEXA fn greet(name: String) -> String { return "Hello " + name + "!" } fn main() { print(greet("World")) print("Code like a Dragon!") for i in 0..5 { print("i = " + str(i)) } }
// Classes in HOLEXA class Dragon { name: String power: Int fn new(name: String, power: Int) -> Dragon { return Dragon { name: name, power: power } } fn roar(self) -> String { return self.name + " roars with power " + str(self.power) } } fn main() { let d = Dragon.new("HOLEXA", 9999) print(d.roar()) }
// Async programming in HOLEXA import http async fn fetchData(url: String) -> String { let res = await http.get(url) return res.body } async fn main() { let data = await fetchData("https://api.example.com") print(data) }
// Error handling in HOLEXA fn divide(a: Int, b: Int) -> Int { if b == 0 { raise Error("Cannot divide by zero") } return a / b } fn main() { try { let result = divide(10, 0) print(result) } catch err { print("Error: " + err.message) } finally { print("Done") } }

Install HOLEXA

One command on any platform

🐧 Linux

curl -fsSL https://raw.githubusercontent.com/HOLEXA-LANG/HOLEXA-LANG/main/install.sh | sh

🍎 macOS

curl -fsSL https://raw.githubusercontent.com/HOLEXA-LANG/HOLEXA-LANG/main/install.sh | sh

🪟 Windows (WSL)

wsl
curl -fsSL .../install.sh | sh

📱 iPad (iSH)

apk add git gcc make
git clone https://github.com/HOLEXA-LANG/HOLEXA-LANG.git
cd HOLEXA-LANG/compiler && make
cp hlxc /usr/local/bin/hlxc

💻 From Source

git clone https://github.com/HOLEXA-LANG/HOLEXA-LANG.git
cd HOLEXA-LANG/compiler
make
sudo cp hlxc /usr/local/bin/hlxc

✅ Verify

hlxc --version
hlxc run hello.hlx
hlxpm --version
