01 Introduction

HOLEXA is a modern, open-source programming language built completely from scratch. It was made for developers who want code that is fast, safe, readable, and simple — all at once.

Most languages make you choose between speed or safety, or between power or simplicity. HOLEXA gives you everything together.

⚡
Fast like C
Compiles to native machine code. Zero overhead. Programs run at full hardware speed.
🛡️
Safe like Rust
No null crashes, no memory leaks. The compiler catches mistakes before they happen.
📖
Readable like Python
Clean beginner-friendly syntax. If you know Python, you can write HOLEXA in minutes.
🔧
Simple like Go
One clear way to do each thing. Fast build. Error messages that help you fix bugs.
🌐
Cross Platform
Linux, macOS, Windows, iPad, Android. Write once, run everywhere.
🐉
Self-Hosting
The HOLEXA compiler is written in HOLEXA itself — proof it is a complete language.

06 Variables & Types

Variables store data in memory. Use let for values that can change later. Use const for values that must never change. HOLEXA figures out the type automatically — you usually don't need to write it.

TypeStoresExampleNotes
IntWhole numberslet x = 42No decimal point
FloatDecimal numberslet f = 3.14Has decimal point
StringTextlet s = "hello"Always in double quotes
Booltrue or false onlylet b = trueOnly two values possible
NoneNo value at alllet n = noneLike null in other languages
variables.hlx
fn main() {
    // Declaring different types
    let name   = "Dragon"     // String
    let age    = 25           // Int
    let height = 5.9          // Float
    let active = true         // Bool
    const MAX  = 100          // Constant — cannot be changed

    // Printing values — numbers need str() conversion
    print("Name: "   + name)
    print("Age: "    + str(age))
    print("Height: " + str(height))
    print("Active: " + str(active))

    // Changing a let variable
    age = age + 1
    print("Next year: " + str(age))

    // Math operators
    print(str(10 + 5))     // 15  — add
    print(str(10 - 3))     // 7   — subtract
    print(str(4  * 5))     // 20  — multiply
    print(str(10 / 3))     // 3   — integer divide
    print(str(10 % 3))     // 1   — remainder
    print(str(2  ** 10))   // 1024 — power
}
Output
Name: Dragon Age: 25 Height: 5.9 Active: true Next year: 26 15 7 20 3 1 1024
⚠️ const cannot be changed. If you try to reassign it, the compiler gives an error.
💡 Always use str() to print numbers. Example: print(str(42)) — not print(42)

08 Functions

A function is a named block of code that does one specific job. You define it once, then call it as many times as you need. Functions make code shorter, cleaner, and easier to fix.

Function Anatomy

PartExampleMeaning
KeywordfnTells HOLEXA you are defining a function
NamegreetWhat you type to call this function
Parametersname: StringInput data the function receives
Return type-> StringType of value the function sends back
Body{ ... }The code that actually runs
Returnreturn valueSends a value back to the caller
functions.hlx
// Function that takes a String and returns a String
fn greet(name: String) -> String {
    return "Hello " + name + "!"
}

// Two Int parameters, returns Int
fn add(a: Int, b: Int) -> Int {
    return a + b
}

// Returns Bool (true/false)
fn is_even(n: Int) -> Bool {
    if n % 2 == 0 { return true }
    return false
}

// No return value — just does something
fn say_hi() {
    print("Hi from HOLEXA!")
}

// Three parameters
fn biggest(a: Int, b: Int, c: Int) -> Int {
    if a >= b and a >= c { return a }
    if b >= a and b >= c { return b }
    return c
}

fn main() {
    print(greet("Dragon"))                // Hello Dragon!
    print(str(add(10, 20)))              // 30
    print(str(is_even(4)))              // true
    print(str(is_even(7)))              // false
    say_hi()                              // Hi from HOLEXA!
    print(str(biggest(5, 12, 8)))     // 12
}
Output
Hello Dragon! 30 true false Hi from HOLEXA! 12
💡 Define functions before main() — or at least before the line where you first call them.

09 Control Flow

Control flow decides which parts of your code run and when. HOLEXA uses if, else if, and else for making decisions.

control.hlx
fn get_grade(score: Int) -> String {
    // Checks from top to bottom — first true one runs
    if score >= 90      { return "A+ — Excellent!"  }
    else if score >= 80 { return "A  — Great!"      }
    else if score >= 70 { return "B  — Good"        }
    else if score >= 60 { return "C  — Average"     }
    else                 { return "F  — Try again"   }
}

fn main() {
    print(get_grade(95))     // A+ — Excellent!
    print(get_grade(82))     // A  — Great!
    print(get_grade(45))     // F  — Try again

    // Using and / or / not
    let x = 42
    if x > 10 and x < 100 {
        print("x is between 10 and 100")
    }
    if x < 0 or x > 1000 {
        print("out of range")
    } else {
        print("x is in range")
    }
    if not (x == 0) { print("x is not zero") }
}
Output
A+ — Excellent! A — Great! F — Try again x is between 10 and 100 x is in range x is not zero

10 Loops

Loops repeat code without writing it multiple times. HOLEXA has two kinds: for (when you know how many times) and while (when you have a condition to check).

loops.hlx
fn main() {
    // for loop — runs i from 0 to 4 (5 is NOT included)
    print("Count up:")
    for i in 0..5 {
        print("  i = " + str(i))
    }

    // while loop — runs as long as condition is true
    print("Countdown:")
    let n = 3
    while n > 0 {
        print("  " + str(n) + "...")
        n = n - 1    // MUST update — or loop never ends!
    }
    print("  Blast off!")

    // Sum 1 to 100 using a loop
    let total = 0
    for i in 1..101 { total = total + i }
    print("Sum 1 to 100 = " + str(total))

    // Nested loops — multiplication table
    for i in 1..4 {
        for j in 1..4 {
            print(str(i) + " x " + str(j) + " = " + str(i*j))
        }
    }
}
Output
Count up: i = 0 i = 1 i = 2 i = 3 i = 4 Countdown: 3... 2... 1... Blast off! Sum 1 to 100 = 5050 1 x 1 = 1 1 x 2 = 2 ... 3 x 3 = 9
⚠️ In a while loop, always update your variable — otherwise the condition never becomes false and the loop runs forever (infinite loop).

11 Classes & Objects

A class is a blueprint — like a cookie cutter. An object is what you make from that blueprint — like an actual cookie. Classes have fields (data they store) and methods (actions they can do).

classes.hlx
// Define the blueprint
class Dragon {
    // Fields — data each object stores
    name:  String
    power: Int
    alive: Bool

    // Constructor — creates a new Dragon object
    fn new(name: String, power: Int) -> Dragon {
        return Dragon { name: name, power: power, alive: true }
    }

    // Method — self means "this object"
    fn roar(self) -> String {
        return self.name + " ROARS! Power = " + str(self.power)
    }

    fn is_legendary(self) -> Bool {
        return self.power > 9000
    }

    fn describe(self) -> String {
        return "Dragon[" + self.name + ", power=" + str(self.power) + "]"
    }
}

fn main() {
    // Create objects from the class blueprint
    let d1 = Dragon.new("HOLEXA", 9999)
    let d2 = Dragon.new("Fire",   5000)

    print(d1.roar())
    print(d2.roar())
    print(d1.describe())
    print("d1 legendary: " + str(d1.is_legendary()))
    print("d2 legendary: " + str(d2.is_legendary()))
}
Output
HOLEXA ROARS! Power = 9999 Fire ROARS! Power = 5000 Dragon[HOLEXA, power=9999] d1 legendary: true d2 legendary: false

12 Error Handling

Errors are normal in programming. HOLEXA uses try and catch so that when something goes wrong, the program handles it gracefully instead of crashing.

errors.hlx
fn divide(a: Int, b: Int) -> Int {
    if b == 0 {
        print("[ERROR] Cannot divide by zero!")
        return 0
    }
    return a / b
}

fn main() {
    // try block — code that might fail goes here
    try {
        print("10 / 2 = " + str(divide(10, 2)))  // works fine
        print("10 / 0 = " + str(divide(10, 0)))  // error — handled
        print("Code still runs after error!")
    } catch err {
        // catch block — runs only if error happened in try
        print("Caught an error: " + err)
    }

    print("Program finished successfully!")
}
Output
10 / 2 = 5 [ERROR] Cannot divide by zero! 10 / 0 = 0 Code still runs after error! Program finished successfully!

14 Recursion

Recursion is when a function calls itself. It needs two things: a base case (when to stop) and a recursive case (calling itself with a smaller input).

recursion.hlx
// factorial: 5! = 5 × 4 × 3 × 2 × 1 = 120
fn factorial(n: Int) -> Int {
    if n <= 1 { return 1 }         // base case — stop here
    return n * factorial(n - 1)   // call itself with smaller n
}

// fibonacci: 0,1,1,2,3,5,8,13,21...
fn fibonacci(n: Int) -> Int {
    if n <= 1 { return n }
    return fibonacci(n - 1) + fibonacci(n - 2)
}

fn main() {
    print("Factorials:")
    for i in 1..8 {
        print(str(i) + "! = " + str(factorial(i)))
    }
    print("Fibonacci:")
    for i in 0..8 {
        print("fib(" + str(i) + ") = " + str(fibonacci(i)))
    }
}
Output
Factorials: 1! = 1 2! = 2 3! = 6 4! = 24 5! = 120 6! = 720 7! = 5040 Fibonacci: fib(0) = 0 fib(1) = 1 fib(7) = 13

import fs

    fs.write("notes.txt", "Line 1\nLine 2\nLine 3")

import subprocess, tempfile, os

def run_holexa(code: str) -> str:
    """Run any HOLEXA code and return its output"""
    with tempfile.NamedTemporaryFile(suffix='.hlx', mode='w', delete=False) as f:

AI Chatbot that Runs HOLEXA

chatbot.py
"""
Chatbot:
1. User asks any math or logic question in plain English
2. OpenAI converts question to HOLEXA code
3. HOLEXA runs the code
4. User sees the answer
"""
import subprocess, tempfile, os
from openai import OpenAI

client = OpenAI(api_key="YOUR_API_KEY")

def ask_and_run(question: str) -> str:
    r = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "system", "content":
                "Convert the question to HOLEXA code. "
                "Only return code. Use fn main(){ } as entry. "
                "Use print() for output."},
            {"role": "user", "content": question}
        ]
    )
    code = r.choices[0].message.content
    with tempfile.NamedTemporaryFile(suffix='.hlx', mode='w', delete=False) as f:
        f.write(code); fname = f.name
    try:
        out = subprocess.run(['hlxc','run',fname],
            capture_output=True, text=True, timeout=10)
        return out.stdout.strip()
    finally:
        os.unlink(fname)

print("HOLEXA AI Bot — ask a math question! (type quit to exit)")
while True:
    q = input("You: ")
    if q.lower() == "quit": break
    print("Answer:", ask_and_run(q))

Number Guessing Game

game.hlx
import random

fn main() {
    let secret = random.int(1, 100)
    let tries  = 0
    let won    = false
    print("Guess a number between 1 and 100! You have 7 tries.")

    while tries < 7 and won == false {
        let guess = int(input("Guess: "))
        tries = tries + 1
        if guess == secret {
            print("🎉 Correct in " + str(tries) + " tries!")
            won = true
        } else if guess < secret {
            print("Too low!  " + str(7 - tries) + " tries left")
        } else {
            print("Too high! " + str(7 - tries) + " tries left")
        }
    }
    if won == false {
        print("Game over! The number was: " + str(secret))
    }
}

Prime Number Checker

primes.hlx
fn is_prime(n: Int) -> Bool {
    if n < 2 { return false }
    if n == 2 { return true }
    if n % 2 == 0 { return false }
    let i = 3
    while i * i <= n {
        if n % i == 0 { return false }
        i = i + 2
    }
    return true
}

fn main() {
    print("Primes up to 50:")
    let count = 0
    for i in 2..51 {
        if is_prime(i) {
            print("  " + str(i))
            count = count + 1
        }
    }
    print("Total: " + str(count))
}

Simple To-Do List App

todo.hlx
import fs

fn load_tasks() -> String {
    if fs.exists("tasks.txt") { return fs.read("tasks.txt") }
    return ""
}

fn add_task(task: String) {
    fs.append("tasks.txt", task + "\n")
    print("✅ Added: " + task)
}

fn main() {
    print("=== HOLEXA To-Do List ===")
    print("Commands: add / show / quit")

    let running = true
    while running {
        let cmd = input("> ")
        if cmd == "quit" {
            running = false
        } else if cmd == "show" {
            print("Your tasks:\n" + load_tasks())
        } else if cmd == "add" {
            let task = input("Task: ")
            add_task(task)
        } else {
            print("Unknown command.")
        }
    }
}
